Création de produit​

POST /api/products/
ItemtypeRequisExemple
nametextYBillet
categorie_articletextYB
send_to_cashlessbooleanNfalse
imgbinaryN/home/jojo/img.jpg

send_to_cashless : Pour indiquer une adhésion associative enregistrée sur carte cashless TiBillet. Un seul produit possible par instance.
